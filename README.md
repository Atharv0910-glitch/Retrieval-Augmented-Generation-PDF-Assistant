# RAG PDF Assistant

An AI-powered Retrieval-Augmented Generation (RAG) application that answers questions from uploaded PDF documents using LangChain, ChromaDB, Hugging Face embeddings, and Ollama.

## Features

- Upload PDF documents
- Semantic search
- Vector database using ChromaDB
- AI-powered question answering
- Local LLM with Ollama

## Tech Stack

- Python
- LangChain
- ChromaDB
- HuggingFace Embeddings
- Ollama
- Streamlit

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
streamlit run app.py
```
